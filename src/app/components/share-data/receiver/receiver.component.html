receivedData: {{ receivedData }}

  • ReceiverComponent has: 

    receivedData: string = '';

  • ReceiverComponent subscribes in ngOnInit() to receive data from CommunicationService: 

    this.commService.currentData$.subscribe((data) => {
     this.receivedData = data; 
    });

  • CommunicationService has: 

    currentData$ = this.dataStream.asObservable()

  • The CommunicationService method transmitData(newMessage: string)
    is used in the SenderComponent to send the newMessage from Sender

    transmitData(newMessage: string) {
      this.dataStream.next(newMessage); 
    }

  • SenderComponent transmits new data via CommunicationService method: 

    publishUpdate() {
     this.commService.transmitData('New Data from Sender!'); 
    }
