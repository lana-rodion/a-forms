In this example, the CartComponent acts as the parent component, which contains the ItemListComponent as a child component.

The ItemListComponent further contains multiple instances of the CartItemComponent as nested child components.

The CheckoutComponent is a sibling component to CartComponent.

The shared service, CartService, is injected into CartComponent, ItemListComponent, and CheckoutComponent, allowing them to interact with the cart data.
